# 🚗 Driver Drowsiness Detection System

## 🔍 Overview
This project implements a real-time driver drowsiness detection system using Python and computer vision techniques. It detects signs of fatigue such as eye closure and alerts the driver to prevent accidents.

---

## 🎯 Objectives
- Detect driver drowsiness in real-time  
- Improve road safety using AI-based monitoring  
- Provide alert mechanisms for fatigue detection  

---

## 🛠️ Tools & Technologies
- Python  
- OpenCV  
- NumPy  
- Machine Learning  

---

## 📊 Features
- Real-time video processing  
- Eye and facial feature detection  
- Drowsiness detection based on eye closure  
- Alert system for driver safety  

---

## 📁 Project Structure
- `driver_drowsiness_detection.ipynb` → Main implementation
